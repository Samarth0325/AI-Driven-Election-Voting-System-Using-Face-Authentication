# AI-Driven-Election-Voting-System-Using-Face-Authentication
This project is an AI-powered secure voting system that uses facial recognition technology to authenticate voters and ensure a transparent and fraud-free election process. The system eliminates the need for traditional ID verification by automatically verifying a voter’s identity through facial recognition before allowing them to cast a vote.
<BR>

🗳️ AI-Driven Election Voting System using Face Authentication
📌 Overview

The AI-Driven Election Voting System using Face Authentication is a secure digital voting platform that leverages Artificial Intelligence and Computer Vision to authenticate voters using facial recognition before allowing them to cast their vote.

Traditional voting systems often face challenges such as identity fraud, duplicate voting, and manual verification delays. This project addresses those issues by integrating AI-based face recognition, ensuring that only verified voters can participate in the election process.

The system captures a voter’s face using a webcam, compares it with stored facial data in the database, and grants voting access only if the identity is verified.

⚙️ System Workflow
1️⃣ Voter Registration

Voters register in the system by providing basic details.

Facial data is captured using a webcam.

The system extracts facial features and stores them in the database.

2️⃣ Face Authentication

When the voter attempts to vote, the system captures a live image.

The AI model compares the captured face with stored face encodings.

If a match is found, the voter is authenticated.

3️⃣ Voting Process

After successful authentication, the voter can select their preferred candidate.

The vote is securely recorded in the database.

4️⃣ Vote Storage

Each voter is allowed to vote only once.

The system tracks voting status to prevent duplicate voting.
